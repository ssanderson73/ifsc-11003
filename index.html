Then I just went back through and really tried to get down all the CSS styling. I really liked this project. Though, it made it both easier and harder when I chose to pick my own topic. I had just finished reading a book that I really enjoyed and had been waiting for. The book is called Alchemised by SinLenYu.

This project was interesting to do and challenged me in so many ways. I am new to all of this so, I went out and got several books and guides for HTMl, CSS, and JS. I also ended up using several online pages to help me. I went line-by-line, back and forth between my monitors, from looking up chapters in my books, and looking up any and everything I could to make what I invisioned MY web page to be. The easiest part I think was getting to content into the html structure. Since we could use help to come up with the content and since I DID read the book, I knew exactly what questions or statements to put in copilot to have it create the content for me. I did read all of it to make sure it was accurate, but that was the easiest part.

One of the hardest things for me, personally, was what I pictured in my head and trying to excute that on github. I got very stuck on SO many things because I wanted and (needed) to recreate what was in my head. I wish I wouldn't have procrastinated this assignment. If I could start back over, I would space it out and plan alot better than I did. I hope what my page ended looking like is okay. I learned a lot from this class and truly enjpyed all of it. I realized I really like doing all the html, css, and JS. Thoughts on Progressive Writing Enchancement

I think that the article gave a great analogy for how poeple should view building a web page. The content is really what everyone comes for, so it needs to accesible for anyone. Web developer's should be staring at the base layer-the structure of the content for the web page- and then building up from that. Progressive enhancement is making sure that the content on web pages are functional and accesible to everyone. My first thought about porgressive enhancement was about houses. You won't be able to suitably survive in a home for very long if it has a bad foundation. You may like the nice fancy kitchen and stove, but you won't like when you're having to redo your floors because the flooring is starting to gap and crack because of a failing foundation. The same goes for a web page. If your core structure, or foundation, is well built first, you can add the styling and details after.
